# UI 组件-加载图

**阅读本文大概需要 5 分钟**

## 什么是加载图？

**加载图**是一种用于制作正提示加载状态动画的组件。

- 变换/对齐/通用/渲染属性请见 [UI 组件的基础属性](https://meta.feishu.cn/wiki/wikcn5pYngyHnkkrJlz8bLMhC9e)
